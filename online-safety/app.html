To install the online safety app -

  1. Download app apk
  2. go to settings
  3. Select Security (Android OS 4.0+) or Applications (Android OS 3.0 and earlier).
  4. You may need to scroll down to see the Unknown Sources setting.
  5. tick the unknown sources button (leave if already checked)
  6. open the downloaded apk file
  7. click install
